## Local development instructions
After cloning this repo
* `$ yarn`
* `cd ios && pod install && cd ..`
* `yarn ios` to run iOS app or `yarn android` to run Android app

### Pre-requisites
* Xcode
* Android Studio and Emulator
* Node
